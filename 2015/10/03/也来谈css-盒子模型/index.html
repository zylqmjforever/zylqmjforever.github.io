概述

－内容(content)、填充(padding)、边框(border)、边界(margin)，CSS盒子模型都具备这些属性
－盒子模型是css中一个重要的概念，理解了盒子模型才能更好的排版。其实盒子模型有两种，分别是 ie 盒子模型和标准 w3c 盒子模型。他们对盒子模型的解释各不相同。
